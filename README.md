# aurora-docs

Architecture, operational guidance and public documentation for the Aurora Defense Network.

Aurora is a distributed defense network designed to protect digital infrastructure
against large-scale attacks such as DDoS, overload events and coordinated abuse.

This repository contains the public-facing documentation for Aurora, including:

- system overview
- architecture
- operational guidance
- installation concepts
- governance documents
- public documentation boundaries

## Current Status

Aurora is currently in active development.
Documentation is being built incrementally alongside the implementation.
